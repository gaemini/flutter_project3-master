Firebase Storage 규칙 수정

다음 주소로 이동:
https://console.firebase.google.com/project/flutterproject3-af322/storage/rules

아래 규칙으로 교체:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read;
      allow write: if request.auth != null;
    }
  }
}

이 규칙의 장점:
✅ 신규 가입자도 즉시 이미지 접근 가능
✅ Firebase 프로젝트 멤버십 불필요
✅ 공개 읽기, 인증된 사용자만 쓰기
✅ 확장성 있는 해결책
